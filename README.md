XBP | Dynamic Test Script Generator
XBP is a high-performance automation tool designed to bridge the gap between technical field specifications and Quality Assurance workflows. It intelligently parses Excel-based data dictionaries and generates standardized, audit-ready test scripts.

🎯 Key Capabilities
Heuristic Header Detection: Uses fuzzy matching to identify header rows within complex Excel structures.

XML-Driven Configuration: Decouples business logic from code, allowing users to map columns via config.xml without script modifications.

Automated Formatting: Applies enterprise-standard styling (cell merging, color-coding, and font weight) to generated outputs.

Validation Logic: Filters and extracts only relevant fields, ensuring clean and focused test data.

🛠 Tech Stack
Core Logic: Python 3

GUI Framework: Tkinter (Custom stylized dashboard)

Data Processing: openpyxl for high-fidelity Excel manipulation

Configuration: XML-based mapping system
